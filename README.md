# Корневой репозиторий ИИ-агента по анализу данных
### [Backend](https://github.com/lowestate/ai_analyst_backend) | [Frontend](https://github.com/lowestate/ai_analyst_frontend)

Загружаете данные, пишете что хотите исследовать, агент визуализирует результат и дает понятное объяснение результатов

## Инструментарий агента:

- Предобработка данных, анализ пропусков и дубликатов по результатам предобработки (автоматически при загрузке даннных)

### Анализ данных
- Исключение столбцов из анализа по требованию пользователя
- Корреляционный анализ:
  1. Корреляционная матрица с фильтрацией корреляции
  2. Подробная зависимость двух признаков
  3. Pairplot с зависимостями всех признаков
  4. Дендрограмма признаков
- Анализ столбцов (распределение, статистики)
- Детекция аномалий
- Тренды
- Важность признаков для определенного признака

### Финансовый анализ
- Движение денежных средств
- Отчет о прибылы и убытках
- Структура расходов
- ABC-анализ (Кумулятивная выручка)
- Юнит-экономика (ARPU vs CAC)
- Прогноз выручки

По графикам чата можно построить дашборд с возможностью экспорта в .png / .pdf
